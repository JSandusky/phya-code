Introduction

Phya is a unique library for generating physically modeled sound in a computer game or simulation environment. It is particularly suited for use with 'physics engines' that can model contact and collision between solid objects. Information from a physics engine can be fed to Phya to generate realistic contact and collision sounds. The acoustic resonance of objects can be included, and this usually dominates the sound generated.

Key features of Phya are:

  1. Logical framework for colliding objects : Bodies, Surfaces, Resonators, Contacts, Impacts.
  2. Efficient modeling of complex scenes consisting of many objects with different resonances and surface properties.
  3. Analysis of real sound samples to create resonance models with small memory footprints.
  4. Variety of parametrised surface types.
  5. Good quality limiter class for preventing distortion on output channels.
  6. Geometric utilities for calculating contact velocities.
  7. Framework designed to be extended with new types of surface and resonator.
